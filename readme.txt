●概要
　MLTをはじめとした(´д｀)Editで読み込み可能なファイルを
　HTMLファイルへと変換するためのコンバータです。

　MLTと名付いていますが、AST、ASDなどのデータも入力可能です。


●動作環境
　Microsoft Windows XP 以上
　Microsoft Internet Explorer 4 以上

　なお、動作環境を満たしている場合でも動作しない場合があるかもしれません。


●基本的な使い方
　MLT2HTML.exe にhtmlにしたいファイルをドラッグ･アンド･ドロップしてください。

　変換したファイルは、元ファイルと同一ディレクトリに生成されます。
　同一ファイル名のものが存在する場合、上書きされますので注意してください。


●使い方(コマンドライン)
　MLT2HTML [-o 出力先ディレクトリ] [-d 変数名 置き換え文字列] 入力ファイル ...

　　-oオプション
　　　出力先に指定されたディレクトリにhtmlファイルを吐き出します。
　　　出力先が不正であった場合、このオプションは無視されます。

　　　また、複数指定された場合は有効な出力先のうち、一番最後に指定されたものが反映されます。

　　-dオプション
　　　template内で置き換えられる変数を追加します。
　　　設定値は「-d 変数名 置き換える文字列」となります。

　　　例えば「-d NAME "ルイズタン ハァハァ"」と指定した場合、
　　　template内で%NAME%となっている場所が、ルイズタン ハァハァと置き換えられます。

　　　このオプションは複数設定することが可能です。
　　　-dオプションで同じ変数を多重に設定した場合
　　　最後に指定したものが使用されます。

　　入力ファイル...
　　　(´д｀)Editで読み込み可能なファイルを指定してください。
　　　複数のファイルを同時に指定可能で
　　　-oオプションで指定したディレクトリ、或いは元ファイルと同じディレクトリに
　　　htmlファイルを出力します。

	生成物の著作権、つまりこのツールによって
　　変換されたファイルの著作権は、元のファイルの著作者に在るものとします。


●テンプレートのライセンスについて
　template以下に存在するファイルはパブリックドメインとして
　上記MLT2HTMLのライセンスに縛られないものとします。

　要するにtemplate以下に存在するファイルは著作権を放棄するので
　お好きにお使いください。
